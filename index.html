Crystal Collector

Rules

There is a target number that the player must match.
There are 4 crystals. At the start of each game, these 4 crystals will get a random value between 1 and 12
You only way for you to know each crystals value is by clicking them and seeing the target number change.
The player wins if their score matches the target number.
The player loses if their score goes above the target number.
The game will restart after an outcome has been achieved
